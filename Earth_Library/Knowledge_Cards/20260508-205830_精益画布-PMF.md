# 一句话定义
精益画布（Lean Canvas）是在精益创业方法论下的一套商业模式速写工具；PMF（Product-Market Fit）则是产品与市场达成契合的状态，由 Benchmark Capital 的 Andy Rachleff 提出，强调对新市场的逐步验证。

# 核心内容

## PMF 四阶段验证路径
| 阶段 | 核心动作 | 关键产出 |
|:---|:---|:---|
| 概念阶段 | 提出早期创意和产品概念 | 概念文档 |
| 原型阶段 | 做出产品外观，让潜在客户试用并反馈 | 可用原型 + 用户反馈 |
| MVP 阶段 | 定义并实现最小可行版本 | 可交付的 MVP |
| PMF 阶段 | 将 MVP 投入市场检验接受度 | 留存率、访问深度、跳出率等多指标判断 |

## 精益画布 vs PMF 的关系
- 精益画布是"画布"（用什么商业模式做）
- PMF 是"验证节奏"（分四步走，逐步确认假设）
- 两者结合：先用画布快速勾勒商业模式，再用 PMF 四阶段逐步验证和迭代

# 使用场景
- 启动新项目/新产品时，用画布快速梳理商业模式假设
- 产品早期阶段，用 PMF 四阶段判断当前处在概念/原型/MVP/PMF 哪一步
- 投资评估时，判断项目是否已经达成 PMF（用留存率等指标交叉验证）

# 与其他框架的关系
- 与 **商业模式画布**：精益画布是其精简化变体（更侧重问题和解决方案，适合早期）
- 与 **5MVVP 框架**：PMF 的阶段划分与 MVP 最小验证一脉相承，PMF 是最终验证目标
- 与 **从产品到产品矩阵**：PSF（问题-方案契合）对应概念/原型阶段，PMF 对应 MVP/PMF 阶段

# 一条边界
PMF 不是"做一次就完成了"的静态状态——市场在变、用户需求在变，已达成 PMF 的产品也会因环境变化而失去契合。**不适用**于已经成熟多年、竞争格局固化的存量市场（此时更适合用波特五力等竞争分析工具）。
