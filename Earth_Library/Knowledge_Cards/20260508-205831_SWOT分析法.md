# 一句话定义
SWOT 分析法从内部（优势 S / 劣势 W）和外部（机会 O / 威胁 T）两个维度分析研究对象，通过单维梳理→复合策略制定，指明战略方向并评估风险。

# 核心内容

## 单维度分析
| 维度 | 性质 | 核心含义 |
|:---|:---|:---|
| S 优势 | 内部有利 | "人无我有，人有我优"——产品/用户/功能/设计等维度的竞争力 |
| W 劣势 | 内部不利 | 不能更好解决用户问题、无价格/运营/体验竞争力等 |
| O 机会 | 外部有利 | 竞品没做或准备做的，（落地后即为优势） |
| T 威胁 | 外部不利 | 产品格局→市场格局→经济格局，逐层升级 |

## 复合维度策略
| 组合 | 策略方向 |
|:---|:---|
| S+O（优势+机会）| 理想型战略，利用优势抓住机会 |
| W+O（劣势+机会）| 做"风口的猪"，用外部机会弥补弱点 |
| S+T（优势+威胁）| 发挥自身优势，规避外部不利 |
| W+T（劣势+威胁）| 防守策略，降低劣势、控制风险、规避威胁 |

## 执行四步
1. 确定分析范围（明确问题背景与目标）
2. 梳理 SWOT 单维度因素
3. 按 S+O → W+O → S+T → W+T 顺序做交叉分析
4. 总结并确定战略方向与行动优先级

# 使用场景
- 企业战略规划
- 竞品分析（分析对手的 SWOT 以制定差异化策略）
- 个人职业规划
- 产品重大转型前做形势盘整

# 与其他框架的关系
- 与 **四大宏观环境分析模型**：SWOT 是其中之一，但本卡有详细的单维+复合方法
- 与 **公司生命周期**：阶段不同，SWOT 中各项的权重应相应调整（如增长期机会权重更高，成熟期威胁权重更高）
- 与 **商业模式画布**：画布描述"现在怎样"，SWOT 分析"可以怎样更好"

# 一条边界
SWOT 分析容易出现"清单式罗列、缺乏优先级"的问题——列出 20 条但每条都给一样重视等于没分析。**不适用**于短期战术决策（如本周迭代排期），此时更适合用 ICE 评分、KANO 模型等更聚焦排序的工具。
