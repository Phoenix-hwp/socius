# 一句话定义
用「问题」和「方案」两个指标的异同构建竞品生态四象限，突破传统「同方案同需求=竞品」的窄视角，发现跨品类竞品和潜在颠覆者。

# 核心内容

## 四象限竞品矩阵

| | 方案同 | 方案异 |
|:---|:---|:---|
| **问题同** | **直接竞品**：用表格对比功能差异 | **潜在颠覆者**：不同方案解决同一问题，帮助理解用户需求本质 |
| **问题异** | **可学习对象**：相同方案解决不同问题，存在跨品类迁移可能 | **资源竞争者**：①占用不可再生资源（时间/金钱/人才）；②产业链上下游未来可能入侵 |

## 关键洞察
- 「问题同方案异」的竞品最容易被忽视——但它们往往是**颠覆巨头的下一代产品**
- 「问题异方案异」看似不相关，但竞争可能来自**用户时间/金钱等不可再生资源**，或**产业链上下游的纵向扩张**
- 分析清楚问题异方案异的竞品，可以将**竞品生态扩展为行业生态**

# 使用场景
- 传统竞品分析做完后，用四象限扩展视野，发现被忽略的潜在威胁
- 评估一个新方向时，检查「问题同方案异」的竞品——它们可能反映了用户需求的更本质解
- 产业链分析时，用「问题异方案异」识别上下游可能入侵的玩家

# 与其他框架的关系
- 与**SWOT 分析法**：四象限可输入 SWOT 的「威胁」和「机会」分析
- 与**B端市场分析与业务调研**：竞品分析是市场分析的重要组成部分——本卡提供分析框架，B端调研卡给出调研方法
- 与**四大宏观环境分析模型**：四象限中的「资源竞争者」角度与 PESTEL 的宏观环境扫描互补

# 一条边界
竞品生态四象限是对传统竞品分析的补充扩展，不是替代——该做的功能对比表（尤其是直接竞品）还是要做。另外，四象限更适合战略层面的竞品扫描，日常功能层面的快速竞品调研可以直接做功能对比即可。
