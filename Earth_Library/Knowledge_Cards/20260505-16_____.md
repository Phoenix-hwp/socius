# Summary
竞品生态的核心要点归纳。

# Details
关于竞品分析，基本的思路还是要【先搞清楚问题】，再【搞清楚解决方案】。通过“问题”和“方案”这两个指标的异同来界定竞品生态的组成。\n- 问题同方案同\n这类属于直接竞品。一般我们用一张表格来对比这些竞品和我方产品的差异。如图：\n- 问题同方案异\n这类是指用不同方案解决相似问题的产品，它们往往会成为行业里颠覆巨头的下一代产品。做竞品分析的时候，要特别关注这类竞品，因为它们可以帮助我们透过解决方案本身，更深刻地理解用户需求的本质到底是什么。\n- 问题异方案同\n这类产品是我们学习的对象。面对用户完全不同的需求，但相同的解决方案可以互相交流学习。如果这样的产品存在跨“品类”迁移的可能，那也就是广义的竞品。\n- 问题异方案异\n问题不同，解决方案也不一样的产品看似不可能是竞品。但也存在两种可能：\n各类竞品都在占用相似的不可再生资源。2016年，罗振宇提出了“国民总时间”的概念，指的是人们的时间已经被各种产品占满了，再要用新的产品，就必须放弃其它某些产品。同样不可再生的资源还有金钱，而对于公司，不可再生的资源也可能是人才。\n问题异方案异的竞争对手还可能来自产品产业链的上下游。任何行业里的某个角色，做大做强后都会占据产业链里更多的位置，以获取更多的利润。所以，分析清楚这类产品，我们也就了解了产品所在的产业链，也可以把竞品生态扩展为行业生态。
