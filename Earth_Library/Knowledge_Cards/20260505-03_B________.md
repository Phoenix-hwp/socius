# Summary
B端产品的演进路线的核心要点归纳。

# Details
B端产品的演进路径中有四个主要阶段：【定制化】、【商业化】、【多元化】、【平台化】。但不是所有B端产品都会经历全部四个阶段。有的产品只会经历其中一两个阶段，而有的头部产品则会经历全部阶段。\n定制化阶段\n定制化阶段是B端产品发展的第一个阶段，也是必经的阶段。\n在这个阶段中，团队根据某一明确业务方提出的明确需求，定制一个仅针对某一具体组织或特定业务场景的产品。\n定制化所针对的组织既可以是本公司（如：企业内部自研自用的产品），也可以是其他组织（如：集团研发部为子公司研发的产品），甚至是一个外包项目。\n- 定制化阶段的产品价值\n解决某一组织内特定场景的问题，满足该组织的业务需求，但此时的产品无法直接产生商业价值，是公司的成本中心，当产品发展逐步成熟，已经能够很好地满足业务需求，经过分析，在其具有足够商业前景之后，就可以考虑将产品推向【商业化阶段】\n注意：虽然有时候可能会看到有的B端产品的创业公司好像没有经过定制化阶段就把产品推向市场了。但其实不是他们没有经历这个过程，而很可能是企业的创始人深耕领域多年，对于该领域的业务痛点、需求非常了解，又或者这个过程之前已经经历了，产品经过了发展和验证，创始人发现了里面的商机才开始创业的。\n定制化阶段的产品是商业化产品的MVP版本。\n商业化阶段\n商业化阶段是产品帮助公司获得外部商业价值的阶段。“商业价值”既可以是真金白银的收入，也可以是市场占有率等。\n在这个阶段是将原来定制化的产品经过改造、升级、包装后，推向市场，实现商业获利。因此，此时的产品不仅要能满足特定且明确场景下的需求，还要具有一定的灵活性，以适应更多组织的情况，满足不同客户的需求。\n产品商业化将产品从成本中心转变为利润中心，对公司来说有极大的好处，但产品商业化需要具备两个必要的前提。\n- 产品在定制化阶段发展得足够成熟。\n判断产品是否足够成熟有以下几个标准：\n- 具有足够的商业前景和价值。\n判断产品是否具有足够的商业前景有以下几个标准：\n1. 有足够多的客户需求。

2. 这个需求能够吸引客户付费。

3. 客户付费所产生的收益大于投入的成本。\n1.\n2.\n3.\n多元化阶段\n多元化阶段是B端产品发展壮大的一个阶段。\n在这个阶段是指产品从原来的仅适应单一或少量场景，向多场景发展的过程。\nB端多元化包括【业务多元化】、【行业多元化】、【部署多元化】和【终端多元化】\n- 业务多元化\n产品从原来的单一的业务领域向多业务领域发展\n- 行业多元化\n产品从原来单一行业解决方案向多行业解决方案发展\n- 部署多元化
