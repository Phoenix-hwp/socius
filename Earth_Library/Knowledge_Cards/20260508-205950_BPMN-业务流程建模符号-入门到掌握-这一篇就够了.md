# 一句话定义
从 0 到 1 掌握 BPMN 的四步路径：定义→四类基础元素（流对象/数据/连接对象/泳道）→实例→绘制，覆盖 BPMN 2.0 核心符号体系。

# 核心内容

## 四类基础元素速查
| 类别 | 子元素 | 核心要点 |
|:---|:---|:---|
| 流对象 | 事件（开始/中间/结束）| 圆圈，控制流程启动、流转与终止 |
|  | 活动（任务/子流程）| 圆角矩形，子流程右下角带 + 号 |
|  | 网关（排他/并行/包容/事件）| 菱形，控制分支与合并逻辑 |
| 数据 | 数据对象、输入/输出、存储 | 建模数据的创建、流转与存储 |
| 连接对象 | 顺序流（实线）| 指定活动执行顺序 |
|  | 消息流（虚线）| 跨池的参与者间通信 |
|  | 关联（点线）| 连接数据/注释与流对象 |
| 泳道 | 池+道 | 池=边界，道=角色 |

## 绘制四步法
1. 套用模板获取 BPMN 符号库
2. 拖拽合适符号到画布
3. 按流程方向和元素关系连线
4. 标注关键元素信息

# 使用场景
- 从零开始学 BPMN，作为快速入门入口
- 实际绘制前对照确认会用到的符号类别
