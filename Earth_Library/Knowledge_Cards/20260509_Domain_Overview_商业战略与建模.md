# 领域地图

## 场景 → 工具速查
| 你想做什么 | 用哪张卡 | 一句话说明 |
|---|---|---|
| 做年度战略规划，先扫一圈大环境 | 四大宏观环境分析模型 | PESTEL→波特五力→SWOT→蓝海战略四模型递进，从宏观到微观、从现状到创新 |
| 判断公司处于哪个阶段，该做什么 | 公司生命周期 | 项目→产品→平台→生态四形态，三大转化路径逐层升级 |
| 设计或检视一个商业模式 | 商业模式画布 | 九宫格工具覆盖客户/价值/渠道/关系/收入/资源/活动/合作/成本 |
| 在精益创业中快速验证假设 | 精益画布 & PMF | 精益画布快速勾勒模式，PMF 四阶段（概念→原型→MVP→PMF）逐步验证 |
| 分析自身或竞品的态势 | SWOT 分析法 | 内部 S/W + 外部 O/T 双维分析，组合出 SO/WO/ST/WT 四类策略 |
| 早期筛选一堆产品点子 | 点子过滤器 | 内部（能力+意愿）× 外部（价值+成本）四维筛选，看"势"是否到位 |

## 常见组合路径
- **路径1**：新市场进入评估 → 先读四大宏观环境分析模型（PESTEL + 波特五力做环境扫描）→ 再读 SWOT 做自我定位 → 用商业模式画布设计进入模式
- **路径2**：创业想法验证 → 先读点子过滤器做初筛 → 再用精益画布 & PMF 做概念验证和逐步迭代
- **路径3**：公司转型判断 → 先读公司生命周期确认当前形态 → 再读商业模式画布检视现有模式 → 用 SWOT 评估转型的优劣势
- **路径4**：竞品全面分析 → 先读 SWOT 做单竞品态势 → 再读四大宏观环境分析模型中的波特五力做行业判断 → 用点子过滤器的外部维度补充评估

# 域内缺口
- **不含"竞品生态"卡**：竞品生态卡在零散域，暂不归入本域；Agent 遇到竞品分析问题时，优先用本域 SWOT + 波特五力，再补读竞品生态卡
- **不含财务预测模型**：本域不覆盖现金流、DCF、ROI 等财务量化分析
- **不含"增长飞轮""网络效应"等平台级理论**：本域偏"分析"和"验证"，不深入平台生态的动力学
