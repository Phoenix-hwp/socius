# 一句话定义
四大宏观环境分析模型（PESTEL、SWOT、波特五力、蓝海战略）是按"从宏观到微观、从现状到创新"递进的战略分析工具集，帮企业理解外部环境变化并制定应对策略。

# 核心内容

## 四大模型速查
| 模型 | 关注什么 | 核心维度 | 最佳用途 |
|:---|:---|:---|:---|
| **PESTEL** | 宏观大环境 | 政治/经济/社会/技术/环境/法律（精简版：PEST） | 市场调研、战略制定前的基础扫描 |
| **SWOT** | 内外部交叉 | 优势/劣势/机会/威胁 + 四象限组合策略 | 态势分析、竞品分析、自我分析 |
| **波特五力** | 行业竞争结构 | 供应商/客户/新进入者/替代品/竞争者 | 行业格局判断、投资评估 |
| **蓝海战略** | 创造新市场 | 同时追求差异化和低成本 | 市场发掘、创新设计、行业颠覆 |

## 递进关系
PESTEL（宏观有什么风吹草动？）→ 波特五力（我所在行业竞争如何？）→ SWOT（我自己优势劣势在哪？面对什么机会威胁？）→ 蓝海战略（能不能不按现有规则玩？）

# 使用场景
- 年度战略规划，先跑 PESTEL + 波特五力做环境扫描
- 新市场进入评估，用五力判断行业是否值得进入
- 竞品分析，用 SWOT 交叉维度分析对手和自己
- 寻找增长第二曲线时，用蓝海战略思维发现未被服务的人群

# 与其他框架的关系
- 与 **SWOT 分析法**：SWOT 是其中之一（本卡为概览），单独的 SWOT 卡片有单维度和复合维度详细拆解
- 与 **点子过滤器**：外部·成本维度直接调用 PESTEL 分析；外部·价值维度与五力分析衔接
- 与 **商业模式画布**：结合使用——先跑 PESTEL + 五力判断赛道，再填画布设计玩法

# 一条边界
这四个模型聚焦**外部环境与竞争**，不直接回答"用户到底要什么"（那是用户研究与产品方法论的事）。**不适用**于已经明确执行路径的短期战术决策——此时用 5W2H 等更聚焦的工具更高效。
