# 一句话定义
商业模式画布是描述和设计商业模式的标准化九宫格工具，覆盖客户、价值、渠道、关系、收入、资源、活动、合作、成本九个维度。

# 核心内容

## 九大模块
| 模块 | 回答的问题 | 要点 |
|:---|:---|:---|
| 客户细分 | 为谁创造价值？ | 按需求、渠道、关系类型、利润率、付费意愿切分 |
| 价值主张 | 解决什么痛点？ | 为特定客户群体创造价值的产品/服务 |
| 渠道通路 | 怎么触达客户？ | 认知→评估→购买→传递→售后 五个阶段 |
| 客户关系 | 维系什么关系？ | 动机：获客/留存/增销 |
| 收入来源 | 怎么赚钱？ | 交易收入（一次性） vs 持续收入（订阅/续费等） |
| 核心资源 | 需要什么资产？ | 实物/金融/知识性/人力，自有或租赁或合作获取 |
| 关键业务 | 做什么最重要？ | 生产/解决方案/平台网络 |
| 重要合作 | 依赖谁？ | 供应商和合作伙伴网络 |
| 成本结构 | 花在哪里？ | 运营商业模式发生的全部成本 |

# 使用场景
- 创业初期，为新业务做商业模式设计
- 分析竞争对手的商业模式，找出差异化和突破口
- 现有业务转型时，用画布重新审视九个模块是否匹配新战略

# 与其他框架的关系
- 与 **精益画布**：商业模式画布是通用完整版，精益画布是其简化版（更适合早期且侧重问题-方案匹配）
- 与 **SWOT 分析法**：画布描述现状，SWOT 分析在此基础上识别优化方向
- 与 **公司生命周期**：不同生命周期阶段，画布各模块的权重和优先级不同

# 一条边界
画布是"静态快照"工具——画完的商业模式只是当前假设，容易让人忽略动态竞争。**不适用**于极度不确定的蓝海探索阶段（此时应先用精益画布 + PMF 验证，再填充完整画布的各模块）。
