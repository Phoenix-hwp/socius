# 一句话定义
支付基础概念的系统索引：从支付定义（付款人→收款人的债权资产转移）→交易/清算/结算三阶段→支付工具/组织/系统/产品→账户体系→会计基础（借贷记账法）。

# 核心内容

## 支付三阶段核心框架

| 阶段 | 定义 | 示例（超市买水） |
|:---|:---|:---|
| 交易 | 确认商品/服务与价格，生成交易单据 | 跟老板确认品牌与价格 |
| 清算 | 数据交换、归集和清分；计算本金和手续费 | 银行卡在POS机刷卡 |
| 结算 | 资金最终实际转移 | 银行账户的钱转移到商户账户 |

## 支付概念速查

| 类别 | 核心概念 |
|:---|:---|
| 支付类型（按业务） | 支付/退款/充值/提现/打款/打款退回/转账/调拨/归集 |
| 支付通道（按类型） | 快捷支付/网关支付/代扣支付/认证支付 |
| 支付方式 | 微信支付/支付宝/银行卡/余额/白条 |
| 收银台类型 | PC/H5/API/硬件/SDK |
| 清结算概念 | 清分/费用项/入账规则/分账/结算周期/轧差/清算文件/结算文件 |
| 账户体系 | 央行清算账户/银行结算账户（ⅠⅡⅢ类）/第三方支付账户（ⅠⅡⅢ类逐级升级）/企业虚拟账户 |

## 支付组织与系统

| 组织 | 核心职能 |
|:---|:---|
| 第三方支付机构 | 收单侧（服务商户）+ 账户侧（服务个人用户，如微信/支付宝） |
| 清算机构 | 银联（银行卡跨行）+ 网联（非银行网络支付） |
| 商业银行 | 行内支付系统 + 银行核心系统（处理资金） |
| 央行 | 最终清算、备付金100%监管、支付基础设施 |

**三大核心系统**：交易系统（订单/账单）→ 支付清结算系统（支付指令→渠道/账务）→ 账务系统（账户/会计）

## 会计基础速览
- **会计恒等式**：资产 = 负债 + 所有者权益；利润 = 收入 - 费用
- **借贷记账法**：有借必有贷，借贷必相等。资产/费用增加记借方；负债/所有者权益/收入增加记贷方
- **会计循环**：编审凭证→编写分录→过账→试算平衡→调整→结账→报表
- **会计科目**：资产类/负债类/共同类/成本类/所有者权益类/损益类
- **账户**：科目是名称，账户是容器

# 使用场景
- 接手支付相关产品时，作为概念速查手册快速定位生僻术语
- 设计支付系统/清结算系统时，参照交易→清算→结算三阶段和账户体系分层
- 跨团队沟通支付需求时，用统一术语（支付类型/通道/方式）避免口径不一

# 与其他框架的关系
- 与**产品服务系统**：支付产品的交付形态演变可参考PSS三类导向（实体/使用/结果导向）
- 与**B端产品市场分析与业务调研**：接入支付产品前需调研目标行业的支付场景和费率结构
- 与**业务架构**：支付系统属于企业IT架构中的核心能力模块
