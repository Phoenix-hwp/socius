# 知识卡片规范

每条知识以独立卡片存储，便于复用、关联和检索。

## 建议字段

- `Title`：知识标题
- `Type`：如读书笔记/网络资料/模型知识/项目结论
- `Source`：来源说明
- `SourceURL`：来源链接（可选）
- `Confidence`：置信度（高/中/低）
- `Keywords`：关键词（逗号分隔）
- `Related`：关联卡片路径列表
- `Summary`：核心摘要
- `Details`：详细内容
- `Next`：后续可扩展问题或行动
