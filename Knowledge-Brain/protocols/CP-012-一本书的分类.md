# 分析阅读一：一本书的分类

## 分析目标

在阅读之前，先确定一本书的**类型**——因为它决定了后续所有阅读规则如何应用。

## 核心判别框架

```
拿到一本书：
1. 它是虚构作品还是论述性作品？
   ├─ 虚构（小说、诗歌、戏剧）→ 用另一套规则
   └─ 论述性 → 继续
2. 它是理论性的还是实用性的？
   ├─ 理论性：告诉你"某事是什么"（是/不是）
   └─ 实用性：告诉你"该怎么做"（应该/不应该）
```

## 理论 vs 实用的判别信号

| 维度 | 理论性作品 | 实用性作品 |
|:---|:---|:---|
| **核心问题** | "这是什么？""为什么会这样？" | "该怎么做？""什么是对的？" |
| **关键词汇** | `是`、`不是`、`存在`、`发生`、`原因` | `应该`、`应当`、`好`、`坏`、`结果`、`意义` |
| **典型门类** | 历史、科学、哲学 | 方法指南、道德论述、工程手册 |
| **检验方式** | 真理（对/错） | 有效（有用/没用） |

## 分类信息来源（优先级从高到低）

| 优先级 | 来源 | 示例 |
|:---:|:---|:---|
| P0 | 书名 + 副标题 | "实用养花指南"→ 实用 |
| P1 | 目录 | 章节以"如何"开头 → 实用 |
| P2 | 序言/引言 | 作者自述写作目的 |
| P3 | 正文关键词扫描 | "应该""应当"密集 → 实用 |

## 应用场景

- 打开任何一本新书之前的第一道判断
- 书架整理：按理论/实用分架
- 选书时快速筛选
