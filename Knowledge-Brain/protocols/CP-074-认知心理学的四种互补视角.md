# CP-074：认知心理学的四种互补视角

## 一、分析目标

理解认知心理学中四种主要理论视角的核心主张、各自优势与局限，以及它们之间的互补关系。

## 二、维度（四种视角）

### 视角 1：信息加工视角

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 认知经过了哪些加工阶段？每个阶段做了什么？ |
| **核心假设** | 认知活动可以分解为一系列阶段；每个阶段对输入信息进行独特加工；最终反应是各阶段操作的结果 |
| **代表性模型** | James 的初级/次级记忆模型、Waugh & Norman 的双阶段记忆模型、Atkinson & Shiffrin 的三阶段模型 |
| **优势** | 直观、可分解、可预测——可以孤立研究每个加工阶段 |
| **局限** | 过于线性，难以解释并行加工、反馈回路和交互作用 |

### 视角 2：神经科学视角

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 认知功能的脑基础是什么？哪些脑区负责哪些认知功能？ |
| **核心争论** | 大脑是整体性器官（功能分布于全脑）还是局部性器官（特定功能有特定脑区定位）？ |
| **关键发现** | Mountcastle 发现皮层神经元以并行排列方式连接——信息加工网络分布于整个大脑皮层，而非局限于少数部位 |
| **研究方式** | 脑外伤个案、单细胞记录、EEG/PET/fMRI/TMS 等成像技术 |
| **优势** | 提供认知的生物实现层面的直接证据 |
| **局限** | "知道哪个脑区亮起来"不等于"理解了这个认知过程怎么运作"——需要信息加工层面和算法层面的解释 |

### 视角 3：计算机科学视角

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 认知过程能否被计算机模拟？模拟能否用来检验和预测认知行为？ |
| **关键贡献** | PDP（平行分布式加工）模型：模仿大脑神经网络的并行加工方式——多个简单加工单元互相发送兴奋/抑制信号，通过联结强度的增强或减弱来学习和存储信息 |
| **核心人物** | Rumelhart & McClelland (1986) |
| **优势** | 可以运行成千上万次模拟验证假设；调整参数直到输出匹配行为数据；然后预测新情境下的行为 |
| **局限** | 计算机和人脑之间有本质差异：人类擅长概括、推测、理解复杂模式、产生情绪；计算机擅长快速运算和遵守逻辑规则 |
| **人机差异** | 面孔识别：人类在毫秒内对陌生面孔做年龄/性别/情绪推测；计算机只能识别已编码的面孔，不能做新异推断 |

### 视角 4：进化心理学视角

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 这个认知过程有什么生存适应功能？它为什么被自然选择保留下来？ |
| **核心前提** | 人类认知的普遍特性是进化所得的心理机制的结果。人类心智进化的环境（99%以上的时间）是狩猎-采集式小规模游牧群社会 |
| **关键人物** | Cosmides & Tooby, Shepard |
| **推理方式** | 既然存在记忆（未被淘汰），它必然具有适应功能 → 追问：记忆如何帮我们生存和适应？ |
| **优势** | 提供了"为什么"层面的解释——给认知过程的功能目的性 |
| **局限** | 容易陷入"故事式"的事后解释；很多认知特征可能是进化的副产品或随机变异，而非被选择的功能 |

## 三、维度间关系

四个视角不是竞争关系，而是**分析层次上的互补**：

```
为什么有这个认知过程？
      ↑
  进化视角：功能追问
      ↑
这个认知过程怎么运作？
      ↑
  信息加工视角：流程分解  ←→  计算机视角：模拟验证
      ↑
这个认知过程用什么硬件？
      ↑
  神经科学视角：脑基础
```

- **进化视角**管"为什么"——最高层的功能目的
- **信息加工视角**和**计算机视角**管"怎么做"——中间层的算法和表征
- **神经科学视角**管"用什么"——底层的生物实现

## 四、应用场景

- 设计认知研究时，明确你要在哪个分析层面提问
- 评估一个认知模型时，检查它是否在四个层面都有解释力（或至少不与其他层面矛盾）
- 理解一个认知障碍时，从四个角度综合判断：哪里受损（神经）、哪个阶段出问题（信息加工）、功能缺陷是什么（进化）、能否用模型复现（计算机）

## 五、边界

- 四个视角不能互相替代：不能用神经科学证据推翻一个信息加工模型（它们在不同的分析层次）
- 进化视角的"功能"解释不等于"当前有用"——有些特征是进化残余而非当前适应

## 六、对系统的启示

- 本系统的四层架构（网关层/能力层/执行层/认知层）与认知心理学的四视角在结构上高度同构：都采用多层次、多视角的互补分析框架
- "为什么-怎么做-用什么"的分析层次，可直接映射到系统审计和决策框架中——遇到复杂问题时同时追问这三个层次
