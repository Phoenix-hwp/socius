## 全局关联 Obsidian 知识库（请复制到 Cursor：设置 → Rules → User Rules）

当 Cursor **当前工作区的根目录**就是 Obsidian 知识库根（例如工作区内存在 `10-Topics/`、`.cursor/rules/long-memory-knowledge-base.mdc` 等）时：

- 若问题涉及持久化上下文、历史约定、项目背景或个人笔记，请在本工作区内用文件工具检索并阅读相关 `.md` 后再回答；引用时使用**仓库内相对路径**（如 `10-Topics/xxx.md`）。
- 若用户要求把结论写入长期记忆，在同一工作区目录树中创建或编辑合适的 Markdown，并遵循库内既有 frontmatter 与目录约定。

**不要**在 User Rules 里写死某台电脑上的盘符绝对路径；换机或移动库后，只要用 Cursor「打开文件夹」选中知识库根目录即可。
