# V006：记忆系统统一

**状态**：vision  
**first_seen**：2026-05-19  
**source**：系统全面评价 — 记忆系统结构性矛盾（7 源分立、LLM 驱动而非 Agent 驱动检索）

## 描述

当前 7 套独立记忆存储（Behavior-Fit-Log / Decision-Log / Round-Behavior-Hot / Round-Behavior-Warm / Active-Task-Tracker / activation-log / 多轮项目 Master_Control）各自独立，LLM 被动检索且常遗漏。远期由 Guard 接管记忆检索：每次任务启动时，Agent 主动从 7 个源拉取相关记忆并聚合注入 LLM#1 上下文，而非等 LLM 自己"想起来"。统一 MemoryStore API：query(task_type, keywords, time_window) → aggregated_context。
