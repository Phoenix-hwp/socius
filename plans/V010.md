# V010：知识脑协议有效性的"生效≠有用"测量

**状态**：vision  
**first_seen**：2026-05-19  
**source**：系统全面评价 — 强注入未落地 + 协议激活不等于有效

## 描述

当前 activation.md 追踪的是协议是否被激活（检索到并注入），但激活 ≠ LLM 实际采纳该协议的指导。远期在 P033 kb_validator 中新增采纳率指标：注入上下文后，检测 LLM 的实际产出是否体现了协议中定义的结构/步骤/原则。若激活率 > 80% 但采纳率 < 20% → 协议可能过于抽象、难以执行、或与 LLM 的推理偏好冲突 → 需要改写或降级。
