# CP-115 多元思维模型与铁锤人倾向

## 一句话定义

**多元思维模型** = 在头脑中构建一个来自不同学科的重要理论的复式框架，用多学科交叉验证替代单一学科的单视角思维。

## 核心命题

1. **铁锤人倾向**：在只有铁锤的人看来，所有问题都特别像一颗钉子。单一学科训练导致的认知盲区是系统性而非偶然性的。

2. **约100种模型**：芒格估计真正重要的思维模型约100种，来自：数学（复利、概率论）、物理学（临界质量）、化学（自我催化）、生物学（现代达尔文综合）、工程学（冗余备份）、心理学（认知误判）、经济学（机会成本、代理成本）等。

3. **复式框架 > 单一模型**：模型之间相互关联，联合使用时产生远大于单独使用的洞察力（→ CP-119 Lollapalooza效应）。

4. **"那我就自己来吧"**：当现有教育体系不提供跨学科训练时，自学是唯一解——芒格从未上过化学/经济学/心理学/商学课程。

## 对 Phoenix 系统的启示

| 启示 | 应用层 |
|:---|:---|
| Guard 的规则体系本身是"铁锤人倾向"的产物——过度依赖 .mdc 文本规则，缺乏工程学/心理学等其他学科的验证 | 架构审计 |
| P008 的 D维度（"跨学科验证"）可直接映射多元思维模型——但当前仅要求"≥2 个视角"，未要求跨学科 | P008 强化 |
| Skills Library 的评估体系可引入"来源学科多样性"作为评分维度 | Skills 评估 |
