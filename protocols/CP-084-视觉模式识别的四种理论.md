# CP-084：视觉模式识别的四种理论

## 一、分析目标

理解大脑如何从视网膜上的二维影像中识别出物体——四种经典理论从简单到复杂递进，描述了模式识别的不同层面。

## 二、维度

### 理论1：模板匹配

| 属性 | 内容 |
|:---|:---|
| **核心机制** | 将输入刺激与记忆中存储的模板逐一比对，找到最佳匹配 |
| **问题** | 无法解释"认出没见过的东西"——新异刺激没有对应模板；也无法解释同一物体的不同视角/光照下的识别 |
| **地位** | 最朴素的理论，已被更复杂的理论替代 |

### 理论2：特征分析

| 属性 | 内容 |
|:---|:---|
| **核心机制** | 将复杂刺激分解为基本特征（线条、角度、曲线等），再组合这些特征识别物体 |
| **优势** | 解释了识字——字母由基本笔画特征组成；解释了为什么相似的字母（P和R）容易混淆——共享特征多 |
| **进阶版**：几何离子理论 | 所有物体都由36个基本的几何离子组成——类似化学中原子组成分子 |

### 理论3：原型匹配

| 属性 | 内容 |
|:---|:---|
| **核心机制** | 大脑存储的不是具体模板，而是某类物体的"最佳代表"（原型）。识别=与多个原型比较，取最接近者 |
| **关键实验** | Posner: 让参与者学习由点组成的三角形→随后呈现从未见过的变形三角形→参与者在更高概率下将其识别为已学三角形→说明他们形成了原型而非记忆具体图形 |
| **对伪记忆的解释** | 大脑会将看到的物体抽象化→原型形成后，具体细节可能丢失→产生"记得但不确定"的感受 |
| **专家效应** | 专家的原型更丰富、更精细（象棋大师存储数万个棋局模式） |

### 理论4：自上而下与自下而上的交战

| 属性 | 内容 |
|:---|:---|
| **核心机制** | 模式识别=自下而上（特征检测→组合→识别）与自上而下（预期、语境、先备知识引导）信息之间的交互 |
| **证据** | 语境效应：同样模糊的图形，在不同语境线索下被识别为不同物体 |
| **专家模式** | 专家=自下而上的特征检测更快 + 自上而下的知识库存更丰富——两者同时加速 |

## 三、典型视角（Canonical Perspectives）

人类对同一类物体的心理表征存在"最佳视角"：
- 搅拌器→正面向左微转，略微俯视
- 杯碟→几乎所有文化的人画出的视角相同（侧视+略微俯视），极少有人画鸟瞰图

这支持了原型理论——我们存储的不是任意视角的模板，而是某个"代表性视角"的原型。

## 四、四种理论的关系

```
模板匹配  →  特征分析  →  原型匹配  →  自上而下+自下而上交互
（天真）      （分解）      （抽象）       （情境化）
```

四种理论不是互相排斥的，而是描述了识别过程的不同层级：
- 特征分析解释了最底层的视觉处理
- 原型匹配解释了分类和泛化
- 自上而下加工解释了语境和专业知识如何加速识别

## 五、对系统的启示

- "分类器-提炼模板"的组合本质上是一次特征分析+原型匹配：classifier提取知识单元的特征(六类判别)→template匹配到最接近的提炼骨架
- 专家模式识别（象棋大师数万棋局模式）可类比到本系统：Agent在多次消化后形成的"领域直觉"——看到一段文字能迅速判断它属于哪类知识、该用什么模板，而无需逐条对照六类判别表
