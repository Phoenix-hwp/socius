# CP-087：长时记忆的类型——程序、语义、情节

## 一、分析目标

理解长时记忆的三大子系统划分及其神经基础——这是记忆研究最重要的分类框架之一。

## 二、总分类框架

```
长时记忆
├── 外显（陈述性）记忆——可以有意识提取
│   ├── 语义记忆：世界知识（概念、词汇、事实）
│   └── 情节记忆：个人经历（自传体事件、时空情景）
└── 内隐（非陈述性）记忆——通过行为表现而非意识
    ├── 程序性记忆：技能、习惯（骑车、打字、书法）
    └── 情绪性记忆：情绪条件反应
```

## 三、三元子系统（Tulving）

### 程序性记忆（最底层）

| 属性 | 内容 |
|:---|:---|
| **神经定位** | 小脑 |
| **内容** | 刺激—反应联结、动作技能、习惯 |
| **提取** | 通过行为表现（内隐），不需要有意识回忆 |
| **遗留现象** | H.M.虽无法形成新回忆，但能学会镜像反转任务——程序性记忆完整 |

### 语义记忆（中间层）

| 属性 | 内容 |
|:---|:---|
| **神经定位** | 大脑皮层 |
| **内容** | 词汇、概念、抽象观念、事实性知识，有组织的知识系统（"心理辞典"）|
| **特征** | 不记录输入信息的知觉属性，只记录认知意义；相对稳定、不易遗忘 |
| **举例** | 知道π≈3.14，知道巴黎是法国首都，知道"狗"是什么 |

### 情节记忆（最顶层）

| 属性 | 内容 |
|:---|:---|
| **神经定位** | 额叶—顶叶—颞叶网络（K.C.案例：左侧额叶—顶叶+右侧顶叶—枕叶损伤）|
| **内容** | 对个人经历事件的记忆（自传体记忆），带有时间/空间背景 |
| **特征** | 容易改变、容易遗失；不断被新的经历检验和更新 |
| **举例** | 还记得第一次去旧金山吃中餐，记得昨天在哪儿停了车 |

### 层级包含关系

```
情节记忆 ⊂ 语义记忆 ⊂ 程序性记忆
```
每个高阶系统依存于低阶系统且有其独特功能：
- 程序性记忆：联结记忆（最低形式）
- 语义记忆额外功能：事物不在面前时能再现内部事件
- 情节记忆额外功能：获取和存储个人经历事件的知识

## 四、关键临床证据：双重分离

### K.C.（车祸，1980）

| 保留 | 丧失 |
|:---|:---|
| 语义记忆（知道消夏小屋位置、会下象棋、知道自己的车）| 情节记忆（无法回忆任何一次在小屋的经历、任何一次下棋、任何一次驾车）|
| 不能回顾"任何一件曾经做过的事物" | 也无法构想未来——被凝固在"不知道过去、无法展望未来"的认知世界 |

### 功能上分离的脑区（PET/rCBF证据）

- **语义记忆任务**→大脑皮层特定区域激活
- **情节记忆任务**→另外区域激活

这证明了塔尔文分类的生理基础——不是理论划分，而是神经事实。

## 五、对系统的启示

| 长时记忆类型 | 系统类比 |
|:---|:---|
| **程序性记忆** | 固化后的规则——被反复使用的 flow-* 规则（Agent "自动"遵守，无需意识调用）|
| **语义记忆** | protocols 目录中的 CP 文件——世界知识，可被查询和引用 |
| **情节记忆** | agent-transcripts 目录中的对话记录——具体经历，带有时间上下文 |

当前系统最缺的是"情节记忆→语义记忆"的提炼通路：对话记录（情节）中反复出现的模式应该被提炼为 protocol（语义）。这就是消化管线的核心价值。
