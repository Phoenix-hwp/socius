# CP-041：产品服务系统（PSS）——实体→使用→结果导向

## 三类产品服务系统

| 类型 | 交易本质 | 典型示例 |
|:---|:---|:---|
| **实体导向** | 买所有权，服务为辅助 | 汽车+保修保养、空调+安装维护 |
| **使用导向** | 买使用权（独占/共享/限时） | 哈啰单车、共享充电宝 |
| **结果导向** | 买结果，实体只是媒介 | 出行服务（不关心工具，只关心到达） |

## 三类系统的演变趋势

| 视角 | 从实体主导到服务主导的变化 |
|:---|:---|
| **用户模式** | 成交终止 → 成交开始（关系更紧密、触点更多、尝试成本更低） |
| **增长模式** | 数量复制 → 人尽其用（未被开发的用户越来越少，精细化运营更重要） |
| **财务模式** | 当期收入 → 预期收入（短期现金流减少，预期收入增加） |

## 产品交付金字塔（按付费量级）

| 付费量级 | 典型交付形态 |
|:---|:---|
| 1元 | 公开引流模块 |
| 10元 | 电子化/实体化工具包 |
| 100元 | 在线课程或图书 |
| 1000元 | 线下大课/线上训练营 |
| 1万元 | 认证课程/高端社群 |
| 10万元 | 2B短期咨询/几天演习 |
| 100万元 | 年度咨询顾问 |
| 1000万元 | 股权期权收益/长期合作 |
