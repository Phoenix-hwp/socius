# CP-088：遗忘的机制与虚假记忆

## 一、艾宾浩斯遗忘曲线

艾宾浩斯（1885）发明无意义音节（辅音—元音—辅音如ZAT、BOK、QUJ），自测遗忘：

| 间隔 | 遗忘比例 | 保持 |
|:---|:---|:---|
| 20分钟 | ~42% | 58% |
| 1小时 | ~56% | 44% |
| 1天 | ~66% | 34% |
| 2天 | ~72% | 28% |
| 31天 | ~79% | 21% |

规律：**前快后慢**——最初几小时遗忘最快，之后趋于平缓。

## 二、遗忘的七种机制

| 机制 | 问题位置 | 核心描述 |
|:---|:---|:---|
| **编码失败** | 输入端（最初就没进来）| 信息根本没进入记忆系统——你在课堂上走神、没注意介绍的名字。耶克斯-道森法则：压力过高/过低都损害编码 |
| **巩固失败** | 转换端（STM→LTM受阻）| 记忆路径形成时被干扰。橄榄球员受伤后30秒能回忆，5分钟后失忆——说明了巩固的时间窗口 |
| **遗忘症（器质性）** | 脑结构损伤 | 阿尔茨海默症（蛋白质绑定谷氨酸抑制记忆）、科萨科夫综合征（B1缺乏）。C.W.：病毒摧毁海马→永远活在1-5分钟的世界 |
| **消退** | 存储端（随时间自然褪去）| 艾宾浩斯曲线描述的现象。但纯消退难以与干扰分离 |
| **干扰** | 存储端（相似记忆混淆）| **倒摄干扰**=新记忆损害旧记忆（换了新柜子忘了旧的）；**前摄干扰**=旧记忆损害新记忆（习惯停老位置找不到新车） |
| **提取失败** | 输出端（线索不匹配）| 编码特异性原则（Tulving）：提取成功=编码时的操作与提取时操作重叠。水下学习→水下提取更好（Godden & Baddeley, 1975） |
| **动机性遗忘/压抑** | 动机端（主动遗忘）| 故意回避创伤经历（动机性遗忘=有意）；无意识推离意识（压抑=无意，弗洛伊德） |

## 三、虚假记忆——记忆可以被建构

### DRM 范式（Roediger & McDermott, 1995）

给参与者呈现：休息/打盹儿/被单/夜晚/床/枕头/梦/打鼾/哈欠...
→ 许多人"记得"看到了"睡觉"——这个词从未呈现，但在语义上与所有词高度联结。

### Loftus 的系列发现

- **引导问题效应**（1974）：问"撞毁"vs"接触"→速度估测相差9英里/小时→甚至"记得"碎玻璃（实际没有）
- **购物中心走失**（1995）：25%参与者"回忆"了从未发生的童年走失事件
- **禁键实验**：69%签署"自首书"承认按了被禁止的按键（9%自创虚构细节）

**核心结论**：记忆不是录像回放，而是每次提取时的**重新建构**。提问措辞、治疗技术、小组分享都能植入虚假记忆。

## 四、对系统的启示

| 遗忘机制 | 系统类比 | 防护 |
|:---|:---|:---|
| 编码失败 | 消化管线中断导致知识未入protocols | Step R 收束确保编码完成 |
| 干扰 | 多个相似protocol混淆 | concept_anchor 区分 |
| 提取失败 | 无法检索到相关protocol | concept-tree + keywords 索引 |
| 虚假记忆 | 被错误消化的知识被当作protocol | P1 判断验证 + P2 四问闸门 |

最危险的系统故障：类似 Loftus 引导问题——如果知识源的措辞偏向性很强（如某段原文本身是立场性论述），系统可能"记住"一个不存在的结论。P1/P2 两级验证正是针对这一点。
