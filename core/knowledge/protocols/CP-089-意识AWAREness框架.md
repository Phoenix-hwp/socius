# CP-089：意识的 AWAREness 框架

## 一、分析目标

理解意识的五大组成成分、科学研究的挑战，以及核心功能——这是认知心理学的终极谜题之一。

## 二、意识定义

**意识是对环境和认知事件的觉察**——包括外部刺激（鸟叫、牙疼、老朋友的脸）和内部心理事件（记忆、思想、情绪、自我感受）。

核心悖论（Damasio）：脑活动可被他人观察，心理活动只有自己可观察。揭示神经基础不会贬低心智——正如揭示玫瑰的分子结构，玫瑰依然芬芳。

## 三、AWAREness 框架（Solso, 2003）

### A — Attention（注意）

| 属性 | 内容 |
|:---|:---|
| **隐喻** | 探照灯——聚焦感兴趣的方向 |
| **机制** | 不断切换注意焦点→切换意识内容。眼睛从视野中一个细节跳向另一个细节 |
| **双向** | 引导向外部线索（海鸟→船只→日光浴者）和内部思想记忆 |

### W — Wakefulness（觉醒）

| 属性 | 内容 |
|:---|:---|
| **本质** | 从睡眠到清醒的连续频谱 |
| **可调控** | 咖啡、冥想、药物均可改变觉醒水平 |
| **与唤醒的关系** | 觉醒≈唤醒（arousal），影响注意 |

### A — Architecture（构筑）

| 属性 | 内容 |
|:---|:---|
| **核心主张** | 意识根植于大脑的生理结构——有其神经机制，具有定位可能性 |
| **已知事实** | 有意识决定（评价电影好坏）和无意识行为（接网球发球）使用不同脑区；语言区主要在左半球 |
| **核心挑战** | 从无意识加工中分离出参与意识的神经活动（Crick & Koch, 1998） |

### R — Recall of Knowledge（知识回忆）

意识使人们能访问三类知识：

| 类型 | 内容 | 测试 |
|:---|:---|:---|
| **自我知识（自我觉察）** | 知道「我是我」。我知道自己在读「意识」这个词 | 镜子测试：2岁后人类通过；黑猩猩、海豚、大象也能通过 |
| **世界知识** | 长时记忆中存储的事实——进博物馆时回忆20世纪艺术 | — |
| **集体知识（共情）** | 知道他人的想法和感受——进化上促进合作生存 | — |

### E — Emotive（感情）

| 属性 | 内容 |
|:---|:---|
| **本质** | 对外部事件的内部情绪反应——脚趾骨折、失恋、意外得A、找到20块钱 |
| **测量难题** | 主观体验几乎无法准确传递给他人——没人能爬进你的头颅连一根管道 |
| **神经影像** | 可观察到沮丧/恋爱时哪些脑区激活 |

## 四、意识的八种功能（Baars & McGovern, 1996）

| 功能 | 描述 |
|:---|:---|
| 情景设定 | 通过情景和记忆中的知识界定输入刺激 |
| 适应和学习 | 成功处理新信息需要意识参与 |
| 优先次序区分 | 从无意识库中获取信息并排优 |
| 召集和控制 | 影响运动系统执行随意行动 |
| 决定和执行 | 从无意识提取资源辅助决策 |
| 错误侦测与编辑 | 深入无意识规则→意识到自己犯错 |
| 自我监视 | 自我反省、内部语言、想象 |
| 组织和灵活性 | 可预测环境用自动功能，不可预测环境用专门知识 |

## 五、全局工作空间理论（Baars, 1988）

**剧院隐喻**：
- 聚光点=意识焦点（工作记忆），被注意探照灯照亮
- 边缘=重要但模糊的意识事件
- 观众=无意识加工系统——从聚光点接收信息
- 幕后活动=塑造聚光点内容的无意识加工
- 意识=在整个大脑中传播信息的**全局广播系统**

## 六、意识状态

| 状态 | 特征 |
|:---|:---|
| **睡眠** | 四阶段+REM；从清醒β波→α波→θ波→δ波（深睡）→REM（做梦）。REM≈清醒的脑活动 + 睡眠的身体 |
| **梦（激活—合成假说）** | REM时的随机脑活动被大脑用长时记忆中的知识框架"解释"为故事。清明梦=知道自己在做梦并可控制 |
| **药物** | 抑制剂（酒精/大麻）、兴奋剂（尼古丁/可卡因）、迷幻剂（LSD）——都作用于神经递质。迷幻剂产生无法被他人验证的主观体验 |
| **冥想** | 从纷繁思绪脱离→放松集中。正面影响左前额叶（激活+理性决策+积极心境）。每天推荐10-30分钟 |

## 七、对系统的启示

- 系统的"意识焦点"=当前消化任务+活跃上下文窗口。Agent的状态切换=探照灯在知识域之间移动
- 大量固化规则（flow-*、mod-*）在"无意识"层面自动运行——只有当这些自动规则产生冲突或错误时，才会被提升到"意识"层级（错误侦测功能）
- Baars的剧院模型可类比：系统规则=幕后活动；当前上下文窗口=聚光点；知识库=观众（被聚光点信息激活相关协议）
