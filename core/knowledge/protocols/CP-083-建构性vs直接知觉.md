# CP-083：建构性知觉 vs 直接知觉

## 一、分析目标

理解人类知觉的两大对立理论——它们各自的核心主张、证据，以及为什么互补而非对立。

## 二、维度

### 建构性知觉（Constructive Perception）

| 属性 | 内容 |
|:---|:---|
| **核心主张** | 知觉=感觉+记忆的主动建构。大脑根据感觉信息提出假设并用经验检验，无意识推论整合多渠道信息 |
| **加工方向** | 自上而下（top-down）——知识/预期驱动知觉 |
| **代表人物** | Helmholtz, Bruner, Gregory, Rock |
| **例子** | 朋友留了胡子、换了发型、戴了墨镜——你仍能认出他，因为大脑用已有知识填补变化的部分 |

### 直接知觉（Direct Perception）

| 属性 | 内容 |
|:---|:---|
| **核心主张** | 刺激本身包含的信息足以产生正确知觉，不需要内部表征、学习或推论 |
| **加工方向** | 自下而上（bottom-up）——刺激驱动知觉 |
| **代表人物** | Gibson |
| **关键证据** | 二战飞行员选拔：深度知觉测试得分高者并不比得分低者更擅长实际飞行→实验室线索不足以解释现实知觉 |
| **美感** | "视觉细胞的繁多恰好与世界的丰富性相匹配"——知觉=环境信息的直接拾取 |

### 结论：互补而非对立

| 理论 | 擅长解释的加工阶段 |
|:---|:---|
| **直接知觉** | 早期感觉检测——刺激信息完整性的重要性；认知是自然、有生态意义的现象 |
| **建构性知觉** | 高级理解——大脑如何从不完整信息中推断意义；对不完全刺激的补全能力 |

## 三、错觉轮廓——建构性知觉的有力证据

卡尼萨三角形：三个缺角圆+三个V形线→物理上不存在的白色三角形"浮现在眼前"。这是大脑用部分信息虚构形状——进化上用于消除伪装、分辨背景中的图形。

## 四、格式塔组织律

| 定律 | 描述 |
|:---|:---|
| **接近律** | 空间上靠近的元素被知觉为一组 |
| **相似律** | 相似的元素被知觉为一组 |
| **闭合律** | 不完整的轮廓被大脑补全为完整形状 |
| **对称律** | 对称的元素被知觉为一组；面孔被知觉为比实际更对称 |
| **连续律** | 按最平滑的路径延续线条 |
| **共同命运律** | 同方向运动的元素被归为一组 |

核心原理（完形律）：大脑倾向于看到简单、熟悉、"良好"的图形。

## 五、对系统的启示

- 系统的"消化"本质上是一个建构性知觉过程：原始文本（感觉输入）+ 分类模板和已有协议（经验和知识）→ 提炼产物（知觉）。不存在纯"直接"的知识消化——任何阅读都已被先前知识着色
- 格式塔闭合律可类比：协议的 `activation.anti_pattern` 字段——看到部分错误信号即触发完整纠正，即使错误并未完全展开
