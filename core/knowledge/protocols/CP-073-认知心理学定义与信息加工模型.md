# CP-073：认知心理学定义与信息加工模型

## 一、分析目标

理解认知心理学的核心定义、研究范围，以及信息加工模型的基本框架。

## 二、核心定义

**认知心理学**是研究人类如何获得、转译、表征、存储和提取知识，以及知识如何引导注意和行为反应的学科。用一句话概括：**认知心理学是关于信息加工的学问**——它研究人们如何注意和获得关于客观世界的信息，这些信息如何被大脑存储和加工，以及如何通过外显行为来证明这些过程。

奈瑟（Neisser, 1967）的定义更为精炼："认知是指将知觉输入进行转换、简化、细化、存储、恢复以及使用的所有过程……每一个心理现象同时都是认知现象。"

## 三、研究范围（维度）

认知心理学覆盖心理过程从输入到输出的全链路：

| 阶段 | 典型研究主题 | 核心问题 |
|:---|:---|:---|
| **感觉与知觉** | 视觉、听觉、错觉、知觉广度 | 外部世界的物理信息如何被转译成心理事件？ |
| **注意与意识** | 选择性注意、自动加工、意识状态 | 有限的加工容量如何分配？什么进入意识？ |
| **模式识别** | 格式塔、模板匹配、特征分析、原型 | 大脑如何从感觉碎片中识别出完整物体？ |
| **记忆** | 短时记忆、长时记忆、遗忘、识记 | 信息如何编码、存储和提取？为什么会遗忘？ |
| **知识表征** | 语义网络、命题、表象、认知地图 | 知识在心理中以什么形式存在？ |
| **语言** | 语法、理解、产生、阅读 | 人类如何理解和产生语言？ |
| **思维与决策** | 概念形成、推理、问题解决、创造力 | 人类如何思考、判断和创造？ |
| **认知发展** | 儿童认知、认知老化 | 认知能力如何随年龄变化？ |
| **人工智能** | 图灵测试、机器视觉、自然语言处理 | 机器能思考吗？认知模型如何被模拟？ |

## 四、信息加工模型的核心假设

信息加工视角的三个基本假设：

1. **序列阶段假设**：认知活动可以分解为一系列阶段（感觉登记 → 注意 → 模式识别 → 记忆存储 → 决策 → 反应）
2. **独立加工假设**：每个阶段对输入的信息进行独一无二的加工
3. **时间累积假设**：最终反应是这一系列阶段和操作在时间上累积的结果

### 典型模型结构

```
外部刺激 → 感觉存储 → 选择性注意 → 模式识别 → 
  短时记忆 ⇄ 长时记忆 → 决策与反应选择 → 行为输出
```

## 五、边界

- **不是神经科学**：认知心理学研究的是信息加工层面的规律（算法/表征层面），神经科学研究的是生物实现层面（硬件层面）。两者互补但分析层次不同。
- **不是行为主义**：行为主义只研究可观察的刺激-反应，认知心理学研究不可直接观察的内部心理过程。
- **不是哲学**：哲学的认知理论依赖思辨，认知心理学依赖可检验的假设和经验数据。

## 六、维度间关系

- 信息加工模型是认知心理学最核心的组织框架，其他研究领域（记忆、语言、思维等）都在这个框架内运作
- 知觉是认知的入口，记忆是认知的仓库，思维是认知的处理器——三者串行、相互依赖

## 七、应用场景

- 需要设计一个认知任务或实验时，用此框架确定研究聚焦于哪个加工阶段
- 需要解释某种认知障碍时，用此框架定位受损阶段
- 需要评估一个 AI 系统的"认知能力"时，用此框架逐一比对

## 八、对系统的启示

- 本系统的「知识消化管线」（Step 0 → 分类 → 提炼 → 评断 → Step R → Step S）本质上就是一个信息加工模型的实例：原始知识 = 外部刺激，分类器 = 模式识别，提炼模板 = 编码存储，Step R/S = 决策与元认知
