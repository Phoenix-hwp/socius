# CP-094：概念形成、逻辑与决策

## 一、概念形成

### 核心定义

概念=一组物体共用特征的集合。概念形成=辨识共同属性的过程。

### 两种主要理论

| 理论 | 机制 | 问题 |
|:---|:---|:---|
| **联想主义** | 反复同时呈现→形成联想；强化奖赏促进。正确配对=强化，错误配对=去强化 | 过于机械，忽略内部结构的主动组织 |
| **假设检验**（Bruner等）| 提出假设→选择样例检验→排除不符→收敛到概念 | — |

### Bruner的四种策略

| 策略 | 运作方式 | 效率 |
|:---|:---|:---|
| 同时性扫描 | 提出所有可能假设→逐一排除 | 低 |
| 继时性扫描 | 一次一个假设，成功则保留 | 中 |
| **保守性聚焦** | 选一个正例为焦点，每次只改变一个特征 | **最有效** |
| 赌胜性聚焦 | 每次改多个特征→更快但更冒险 | 中 |

## 二、演绎推理——三段论

### 基本结构

```
大前提：所有的A都是B
小前提：所有的C都是A
结论：所有的C都是B（有效）
```

256种可能组合中仅24种逻辑有效。

### 三种系统性偏差

| 偏差 | 描述 | 例子 |
|:---|:---|:---|
| **气氛效应** | 推理形式影响可信度——词汇替换改变"气氛" | "所有的共和党人都是人类/所有的民主党人都是人类/所以共和党人=民主党人"→识字都知错，但形式相同用抽象符号时不易察觉 |
| **内容偏向** | 结论与已有态度一致时更容易接受（即使逻辑无效）| "砷不是苦的/有毒的东西是苦的/所以砷不是有毒的"—人们因为不希望砷有毒而接受错误结论 |
| **形象性与关联性** | 高形象性+高关联性的前提更容易正确推理 | "男人想要油炸圈饼→去面包店">"女人重组公司结构" |

### Johnson-Laird 心理模型理论

人们推理不是通过形式规则，而是通过**建构情境模型**。用图示（路径+插槽）呈现推理问题→正确率比纯文字提高约30%。

## 三、决策——启发式与偏差

### 代表性启发式

人们根据"与典型有多像"来判断概率，忽略基础比率。

### 决策框架效应

同一选项用"拯救200人"vs"失去400人"措辞→选择反转。损失框架下人们更愿意冒险。

### 贝叶斯定理核心直觉

先验概率 × 新证据 = 后验概率。人们通常忽略基础比率而过度依赖个案信息。

## 四、常见谬误论证

| 谬误 | 特征 |
|:---|:---|
| 具体化造成的假象 | 把抽象当实体 |
| 对人不对事 | 攻击人而非法论点 |
| 诉诸权威/名望 | 权威说的就是对的 |
| 稻草人论证 | 歪曲对方论点→攻击歪曲版 |
