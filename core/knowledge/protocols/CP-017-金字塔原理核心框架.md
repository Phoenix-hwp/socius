# CP-017：金字塔原理核心框架

**Type**：框架（Framework）  
**Domain**：通用思维方法  
**Created**：2026-05  

## 概述

来自 Barbara Minto 的《金字塔原理》：任何思想都可以归纳成一个中心论点，由 3-7 个论据支持；每个论据本身又可以是一个论点，向下延伸形成金字塔结构。

## 核心原则

1. **结论先行**：每层只有一个中心思想
2. **以上统下**：上层是对下层的概括
3. **归类分组**：每组中的思想属于同一逻辑范畴
4. **逻辑递进**：每组中的思想按逻辑顺序排列

## 应用场景

- 写作与演示：SCQA 序言结构（情境→冲突→问题→回答）
- 问题分析：自上而下构建论点金字塔
- 系统设计：模块分层遵循归类分组原则
