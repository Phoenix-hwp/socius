# CP-040：产品关键指标制定三原则

## 三条制定原则

### 原则 1：事前制定，不可事后调整

- 项目执行前定好；无重大变化不调整
- 提前想清楚退出机制和终止投入条件
- 反模式：过程中发现完不成就「灵活调整」指标

### 原则 2：不能太虚也不能太实

- 太虚难考核 → 用 SMART 原则兜底（具体/可衡量/可达到/与目标相关/有时间限制）
- 太实易作弊 → 用「防作弊三条」兜底

### 原则 3：防作弊三条

| 条 | 含义 |
|:---|:---|
| **非受迫** | 用户不被逼做无价值动作（如不注册就不能看） |
| **无诱导** | 非奖励利益驱动（如签到领积分=虚假活跃） |
| **成功行为** | 行为本身为用户创造价值，非只对公司有价值 |

## 基石：对齐基线

指标制定须与业务目标、产品使命愿景价值观相契合——否则指标再 SMART 也只是在错误方向上跑得更快。

## 边界

- 本卡只提供指标制定的**原则框架**，不提供各阶段的具体指标清单
- 实际操作需结合产品生命周期阶段和业务类型自行定义
- 防作弊三条是必要条件非充分条件——通过三条不代表就是好指标
