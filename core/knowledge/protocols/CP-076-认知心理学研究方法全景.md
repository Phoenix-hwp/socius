# CP-076：认知心理学研究方法全景

## 一、分析目标

系统地理解认知心理学中可用的研究工具：每类方法的核心问题、分析单元、关键假设和典型发现。

## 二、总分类

认知研究方法分为两大类：

| 大类 | 核心功能 | 方法清单 |
|:---|:---|:---|
| **心理与物理世界关联的测定** | 测量个体对外部事件的反应——确定因果关系 | 心理物理学、单细胞研究、反应时研究、启动效应、眼动研究、偏侧化研究 |
| **独特案例的记录** | 观察自然发生的损伤或极端个体——发现线索、形成假说 | 个案研究、成像研究 |

## 三、方法详解

### 类别一：心理与物理世界关联的测定（实验法）

#### 3.1 心理物理学

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 物理刺激与心理感知之间的函数关系是什么？ |
| **分析单元** | 个体（通常是研究者自己） |
| **关键发现** | 韦伯的最小可觉差——需要增加多少物理刺激人才能察觉差异，这种增量遵循数学规律 |
| **关键假设** | 人类以相似的方式感知，因此测量自己的知觉经验可以推广 |
| **历史地位** | 心理学中最古老的方法之一 |

#### 3.2 单细胞研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 特定神经元对特定刺激是否有选择性反应？ |
| **操作** | 用微电极（比头发丝还细）插入单个神经元，测量电活动 |
| **分析单元** | 单个细胞 |
| **关键发现** | Hubel & Wiesel (1959) 发现猫视觉皮层细胞对特定方向的光条有选择性反应 → 诺贝尔奖 |
| **关键假设** | 如果某细胞对某刺激有反应，该细胞与该刺激的加工有关 |
| **限制** | 侵入性操作，主要在动物身上进行 |

#### 3.3 反应时研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 不同认知操作各需要多少时间？用时间差推断加工阶段 |
| **经典实验** | Donders：简单反应（见光按键）vs 选择反应（见黄光才按键）→ 时间差 = 辨别所需额外加工 |
| **另一个经典** | Shepard：心理旋转 45° 比 15° 花更多时间 → 心理事件对应物理世界的性质 |
| **关键假设** | 认知活动需要时间；一个阶段结束下一阶段才开始（序列加工） |
| **历史地位** | 认知心理学产生的标志性方法 |

#### 3.4 启动效应研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 先呈现的刺激如何影响后续刺激的加工？ |
| **两种类型** | 语义启动（先见"医生"→ 更快识别"护士"）+ 客体启动（先见飞机素描 → 更好识别退化后的飞机轮廓） |
| **关键假设** | 认知网络中被激活的知识使相关知识的提取变得更容易 |
| **技术演进** | 从速示器到计算机到脑成像——方法不断升级但核心逻辑不变 |

#### 3.5 眼动研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 个体每时每刻在看什么？注意如何分配？ |
| **关键应用** | 阅读研究——准确判断注视哪个词、下一个词、停留时间 |
| **发现** | 有阅读障碍者与正常读者有不同的眼动模式 |
| **关键假设** | 大脑指示眼睛朝向对认知功能重要的位置 |

#### 3.6 偏侧化研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 大脑两半球各自负责什么认知功能？ |
| **非侵入法** | 在左/右视野快速呈现刺激（左视野→右脑、右视野→左脑）→ 比较反应时 |
| **侵入法（裂脑研究）** | Gazzaniga 对切断胼胝体的癫痫患者做单侧刺激呈现 → 两半球不交流时研究各自的功能 |
| **关键假设** | 信息通过胼胝体在两半球间传递需要额外时间；切断后两半球独立工作 |

### 类别二：独特案例的记录（观察法）

#### 3.7 个案研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 脑损伤导致什么认知缺陷？缺陷与受损部位的关系？ |
| **经典案例** | 布洛卡区损伤 → 言语产生缺陷；威尔尼克区损伤 → 言语理解缺陷 |
| **关键假设** | 特定功能有特定脑区定位 |
| **限制** | 个案不是实验——无法确定因果关系；损伤范围往往不精确 |

#### 3.8 成像研究

| 属性 | 内容 |
|:---|:---|
| **核心问题** | 认知活动时哪个脑区活跃？结构和功能分别是什么样？ |
| **分类** | 结构成像（显示神经损伤的细节）+ 过程成像（显示活动的时间和位置）+ 复合成像 |
| **常用技术** | EEG、CT、PET、MRI/fMRI、MEG、TMS |
| **关键假设** | 活跃脑区需要更多氧气、产生更多代谢废物 → 更多血液流向该区域 → 可被成像技术检测 |

## 四、方法概念基础

### 4.1 操作性定义

- **概念性定义**：对概念的日常理解（如"醉酒"）
- **操作性定义**：将抽象概念转化为可测量的具体指标（如"血液酒精浓度 >= 0.08"）
- **为什么重要**：没有操作性定义就无法做实验——你没法测量一个模糊的概念

### 4.2 分析单元

- **定义**：研究的焦点对象——你实际在测量什么
- **常见单元**：个体（认知心理学最常用）、细胞（单细胞研究）、群体（社会心理学）、对话、城市、人工制品
- **原则**：澄清你使用什么分析单元是理解任何研究的第一步

## 五、维度间关系

- **实验法 vs 观察法**：实验法可以确定因果关系（操纵自变量→观察因变量），观察法只能发现关联
- **行为 vs 生理**：反应时/启动/眼动/心理物理测量行为层面，单细胞/偏侧化/成像测量生理层面
- **时间 vs 空间分辨率**：不同成像技术各有侧重（EEG 时间精度高空间精度低，fMRI 相反）

## 六、应用场景

- 评估一篇认知研究论文时，首先辨认其使用的方法及其假设
- 设计新研究时，根据研究问题选择方法——要因果找实验法，要脑基础找成像
- 理解看似矛盾的研究结论时，检查两者是否用了不同的分析方法或操作性定义

## 七、边界

- 个案研究不能替代实验——它提供线索但不能确立因果关系
- 成像技术告诉你"哪里"活跃，不告诉你"为什么"——需要信息加工理论来补充
- 不同方法的假设不同，不能简单将不同方法的结果直接对比
