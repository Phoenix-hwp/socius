# V009：令牌制经验系统

**状态**：vision  
**first_seen**：2026-05-19  
**source**：系统全面评价 — 经验被 LLM 随机忽略（简历修改案例）

## 描述

当前经验/记忆的检索是 LLM 驱动的——LLM 需要主动去读 Behavior-Fit-Log / Decision-Log 等文件，但它经常完全忽略经验和记忆。远期由 Guard 在每次任务启动时：1) 根据任务类型检索相关历史经验；2) 将经验浓缩为 ≤3 条令牌注入到 LLM#1 的 system prompt 中；3) 令牌格式：【上次同类任务 X 选择了方案 A，结果 Y】【历史偏好：用户倾向于 Z 风格】。经验不再是被动的文件，而是主动注入的令牌。
