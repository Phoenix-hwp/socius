# Socius 使用指南

> 你的 AI 工作搭档 — 自主认知 · 自决策 · 自执行

---

## 快速开始

直接描述你想做的事，我会自己判断怎么执行：

| 你想做的事 | 直接说 |
|:---|:---|
| 查看任务清单 | `任务清单` / `task list` |
| 查看待办 | `待办` / `decision` |
| 查看旧样式待办 | `todo` / `list`（兼容旧习惯） |
| 创建任务 | `创建一个任务：整理项目文档` |
| 分解任务 | `帮我分解任务：搭建登录模块` |
| 读取文件 | `读取 README.md` / `看看 plans/ 目录` |
| 搜索代码 | `搜一下在哪里定义了 ContextBuilder` |
| 写文档 | `写一份本周工作总结` |
| 执行命令 | `帮我跑 npm install` / `查看 git 状态` |
| 代码审查 | `审查 socius_cli.py 的代码质量` |

**不需要记固定指令——怎么说都行。**

## 🔥 亮点功能

### 复杂任务自主拆解排期

跟我说「重构认证模块」，我会自动把它拆成 5–10 个子任务，排好先后顺序，逐个推进。你不用操心先做什么后做什么。

**试试**：在 `> ` 提示符下输入：
```
帮我分解任务：重构用户认证模块
```

### 待办自决策自执行

给我一句「帮我优化首页性能」，我就自己读代码、自己找瓶颈、自己改——你只需要验收结果。

**试试**：在 `> ` 提示符下输入：
```
创建一个待办：检查项目中是否存在硬编码路径
```

## 会话控制

| 操作 | 怎么做 |
|:---|:---|
| 退出 Socius | 输入 `q` / `exit` / `/quit` |
| 终止当前任务 | 按 `Ctrl+B` |
| 查看帮助 | 输入 `help` |
| 调试模式 | 启动时加 `--debug`：`socius --debug` |

---

## CLI 命令

| 命令 | 用途 |
|:---|:---|
| `socius` | 启动交互模式 |
| `socius --debug` | 启动交互模式（显示模型每轮输出） |
| `socius list-models` | 列出已注册的模型提供商 |
| `socius verify` | 框架层自检 |
| `socius run --prompt "..."` | 单次任务执行 |
| `socius --help` | 查看 CLI 帮助 |

---

## 模型配置

Socius 首次启动会引导你选择模型提供商和输入模型名称。支持：

- **本地模型**：Ollama / LM Studio（免费，无需 API Key）
- **云端模型**：OpenAI / Anthropic / DeepSeek / Kimi（需 API Key）
- **自定义端点**：任何 OpenAI 兼容的 API

API Key 在输入时会显示 `*` 回显，仅保存在当前会话中，关闭终端后自动清除。
