# AI Model Shim - Binary Packages

## ngrok.exe

**Version**: 3.x.x (Latest Stable)
**Size**: ~20MB

**Download** (if USB not available):
https://ngrok.com/download
→ Download for Windows (ngrok.exe)

**Config** (one-time):
```
ngrok authtoken YOUR_TOKEN
```
Get token from: https://dashboard.ngrok.com/get-started/your-authtoken

---

## How to Deploy

1. Copy `ngrok.exe` to your repo: `<repo>\.cursor\ai-model-shim\`
2. Run `bootstrap-on-pull.cmd` from repo root to complete setup
3. Ready!
